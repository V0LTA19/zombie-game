# Kıyamet Operasyonları

Tarayıcı tabanlı RPG shooter envanter, artı basma, rütbe ve level sistemi prototipi.

## Yerelde çalıştırma

```bash
npm install
npm run dev
```

Sonra tarayıcıdan aç:

```text
http://localhost:3000
```

## Vercel'e yükleme

1. Bu klasörü GitHub'a yükle.
2. Vercel hesabına gir.
3. Add New Project > GitHub reposunu seç.
4. Framework otomatik Next.js görünecek.
5. Deploy butonuna bas.

Deploy sonrası sana `https://proje-adi.vercel.app` benzeri bir link verir.
